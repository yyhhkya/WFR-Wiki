# 常见问题

## 神秘箱

❓饰品卡物品栏丢不出去

✅当使用饰品中，请勿拿着原版装备右键快捷穿戴，这会导致饰品被替换到物品栏且无法恢复。（不小心卡了只能找OP恢复）

## 领地

❓我缩小领地为什么还要扣钱？

✅缩小领地并不会扣钱。假如你把10格缩小15格，结果为-5，那就是反向增加了5格。请不要让缩小的数大于原来的数。
